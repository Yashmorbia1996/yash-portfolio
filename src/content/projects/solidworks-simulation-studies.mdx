## Tools & Capabilities

| Tool | Use |
|------|-----|
| SolidWorks Flow Simulation | Internal flow, velocity distribution, pressure drop, mass flow balance, porous media approximation |
| Porous Media Modelling | Catalyst block approximation using isotropic resistance, efficient for parametric design sweeps |
| Goal Tracking | Equation goals for uniformity ratio and pressure drop, Averaged Value column for design comparison |
| Post-processing | Velocity/pressure cut plots, front-face velocity distributions, flow trajectories coloured by velocity |
