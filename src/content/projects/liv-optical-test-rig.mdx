{/* THE PROBLEM */}

<div id="the-problem" class="not-prose mb-16 scroll-mt-28">
  <div class="mb-6">
    <h2 class="text-4xl font-semibold tracking-[-0.04em] text-text-primary md:text-5xl lg:text-6xl">
      The problem
    </h2>
    <h3 class="mt-4 text-2xl font-semibold tracking-[-0.03em] text-text-primary md:text-3xl">
      Coupling and alignment error masquerading as device failure
    </h3>
    <p class="mt-4 max-w-3xl text-base leading-relaxed text-text-secondary">
      <strong class="text-text-primary">LIV</strong> relates optical output to drive current and terminal voltage. The measurement is only meaningful if the <strong class="text-text-primary">optical coupling</strong> between the engine and the collection path is stable, centered, and repeatable across load/unload. Sub-micron-level <strong class="text-text-primary">tilt, decenter, or piston</strong> in the fixture can change coupled power into the sphere or detector — producing <strong class="text-text-primary">apparent</strong> LIV shifts that look like emitter drift or marginal parts.
    </p>
    <p class="mt-4 max-w-3xl text-base leading-relaxed text-text-secondary">
      In a regulated screening context, that failure mode is especially damaging: a <strong class="text-text-primary">false negative</strong> can let a misaligned run look “in spec,” while inconsistency can drive <strong class="text-text-primary">false scrap</strong> or unnecessary rework. The legacy flow was <strong class="text-text-primary">manual, slow, and operator-sensitive</strong>, which is incompatible with high-volume batch release from the clean room.
    </p>
    <p class="mt-4 max-w-3xl text-base leading-relaxed text-text-secondary">
      The program needed <strong class="text-text-primary">datum-defined seating</strong>, <strong class="text-text-primary">kinematic or equivalent constraint</strong> where appropriate, <strong class="text-text-primary">tolerance analysis</strong> on the optical mechanical stack, and <strong class="text-text-primary">released drawings</strong> under ISO 13485–style design controls so inspection and production could execute the same configuration every time.
    </p>
  </div>
</div>
